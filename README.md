# Blender
A "LLM knowledge blender" that breakdown and organize knowledge to a way that is easy to be digested. Everyone digest knowledge differently, this blender can learn and adapt through communication with you.

## Design

Blender is a LLM based personal knowledge base, inspired by the [LLM Wiki by Andrej Karpathy](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f). The most important productivity improvement from the AI revolution is the speed to learn. With help of AI, we can, and we have to if we want to keep up, learn things 10x faster.
